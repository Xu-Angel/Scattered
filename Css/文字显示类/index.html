对于一个元素，如果其display属性值是inline-block，那么其宽度由内部元素决定，但永远小于“包含块”容器的尺寸，也就是“包裹性（shrink-to-fit）” 当文本比较少时，.text的宽度跟随文本，然后我们可以使用父级text-align:center来使一个inline-block元素居中，所以可以满足单行文本居中的效果， 当文本比较多时，.text的宽度跟随父级容器，由于text-align:center会继承下去，所以在.text上修复一下即可 作者：XboxYan 链接：https://juejin.im/post/5e01bf03e51d45581b11f376 来源：掘金 著作权归作者所有。商业转载请联系作者获得授权，非商业转载请注明出处。
